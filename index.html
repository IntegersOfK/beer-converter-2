What's new

A high-level tour of features that have shipped, newest first.

Shared sessions
  • Sessions now live on the server, so everyone with the link contributes to the same tally.
  • Copy the current session link from the tally header.
  • Final reports load live from the session and keep polling for updates.
Quick-add polish
  • "Same again" button next to every person's add control.
  • Most-recently-used drink types float to the front of the chip tray.
  • "Save this as a drink type" defaults to on.
  • Refreshed header icons — calendar for sessions, cocktail glass for drink types.
Per-drink flavours
  • Curated products can carry multiple flavours under one name (e.g. White Claw · Mango).
  • Edit a logged drink's flavour without touching the saved drink type.
  • Flavours render on drink rows and on the shareable final report.
Editing & sessions
  • Edit any logged drink — apply the change to just that one or to every drink of the same type.
  • Switch between past bar trips, rename them, see who was there.
Look & feel
  • Bar (dark) / Beach (light) theme toggle for sun-readable patios.
  • Global ml / oz unit preference.
  • Tooltips on every number and badge.
Final report
  • Shareable report with donut chart and per-person breakdown.
  • Report links point at the live session instead of a copied payload.
Catalogue & barcodes
  • The shared product catalogue now uses SQLite behind the scenes.
  • Scan a UPC against the bundled BC Liquor product list.
  • Unknown barcodes get submitted for review and folded into the shared catalogue.
Foundations
  • N-person tally with compare-everyone detail panel.
  • Standard-drink math (1 std drink = 17.05 ml ethanol) against Health Canada guidelines.
